# pfrederiksen Homebrew Tap

Install the tap:

```bash
brew tap pfrederiksen/tap
```

Install formulas:

```bash
brew install agent-scope-diff
brew install openclaw-cron-lint
```

Available Python CLI formulas:

| Formula | Description |
| --- | --- |
| `agent-scope-diff` | Risk-focused diffs for agent manifests and tool permission snapshots. |
| `openclaw-cron-lint` | Lints OpenClaw cron job configurations for risky delivery and prompt patterns. |

Both formulas install from tagged GitHub source archives and use Homebrew-managed Python virtual environments.
